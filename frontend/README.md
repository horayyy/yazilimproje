# Hospital Appointment System - Frontend

React + Vite frontend application for the Hospital Appointment System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Dependencies

- **React** - UI library
- **Vite** - Build tool
- **React Router DOM** - Navigation
- **Axios** - API requests
- **JWT Decode** - Token decoding
- **Tailwind CSS** - Styling

## API Configuration

The frontend connects to the Django backend API at `http://127.0.0.1:8000/api/`
